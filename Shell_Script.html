Cheatsheet Of Shell Scripting!
  1. Steps to write and execute a script: 
    1. Open the terminal. Go to the directory where you want to create your script.
2. Create a file with .sh extension.
3. Write the script in the file using an editor.
4. Make the script executable with command chmod +x <fileName>.
5. Run the script using ./<fileName>.
  2. Comments: 
    #This is comment.
  3. Print: 
    name = "Hello World!!!"
echo $name
  4. variable: No need to specify the type of variable. 
    var1 = 23
var2 = 35.45
var3 = "sonu mahajan"
  5. Comparison Expressions: 
    >   -gt
<   -lt 
>=  -ge 
<=  -le 
==  -eq 
!=  -ne
  6. Input: 
    read x
echo $x
  7. if else Statement: 
    echo "Eneter any number: "
read no
if [ $no -gt 100 ]
then
echo "$no is greater then 100"
else
echo "$no is less then 100"
fi
  8. Nested if else Statement: 
    echo "Eneter any number: "
read no
if [ $no -gt 100 ]
then
        echo "$no is greater then 100"
elif [ $no == 50 ]
then
        echo "$no is half of 100"
else
        echo "$no is less then 100"
fi
  9. Case Statement: 
    read a
case $a in
        10) echo "Ten";;
        20) echo "Twenty";;
        30) echo "Thirty";;
        *) echo "Random number";;
esac
  10. While loop: 
    x=1
while [ $x -le 10 ]
do 
echo $x
x=`expr $x + 1`
done
  11. Delay Output: 
    x=1
while [ $x -le 10 ]
do 
echo $x
x=`expr $x + 1`
sleep 1
done
  12. For Loop(1): 
    for((x=1;x<=10;x++))
do 
echo $x
done
  13. For Loop(2): 
    for t in 1 2 3 4
do 
echo $t 
done
  14. For Loop(3): 
    date
for t in 'date'
do 
echo $t 
done
  15. Until Loop: 
    x=1
until [ $x -lt 10]
do 
echo $x
x=`expr $x + 1`
done
